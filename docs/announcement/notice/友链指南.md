# 📍友链投稿指北🧭
1. 友链目前采用github工作流形式自动添加，所以你需要准备1个github账号（这不废话吗）

2. 工作流通过检测PR所提交的修改来自动打上友链的TAG并执行下列工作流，你需要修改.vitepress\theme\untils\data.ts中的内容，并且不要与其他文件同时提交，1次提交只能提交1个友链！！！否则工作流无法正常运行

3. 当工作流检测到你只修改了.vitepress\theme\untils\data.ts中的内容并且格式正确后，会依次为你打上紫色/红色TAG（紫色为验证成功，红色为出现问题），如果你的友链被打上红色标签，虽然工作流不会立即关闭你的PR，但建议你主动关闭PR修改后提交，否则后期可能会被我手动踹下台😡

4. 在验证所有权阶段中，其原理是通过访问你网站根目录的一个名为qingyun-verify.txt的文件，看看是不是404，不是就直接合并，但如果你的网站被打上“网站不可达”的标签，那么所有权验证阶段大概率就会被失败

5. 一些小提示：link和icon都需要包含 http:// ，不能直接填写域名

6. 请补药恶意上传一些黄赌毒网站

7. 如果你发现超过5分钟github的工作流还没有自动运行请自行对照（2）中的指北小妙招看看自己有没有中招，没有就私聊群里面任意一位有仓库协作的手动合并

**🔗仓库链接：https://github.com/Explore114/qingyun_doc**
